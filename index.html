Aurora Web Framework

Golang Web

快速上手 →

开箱即用

使用简单,无需繁琐配置,急速上手。

零断言

处理请求的接口，直接解析请求参数，无需断言进行参数转化。

package main

import (
	"fmt"
	"github.com/aurora-go/aurora"
)

func main() {
	web := aurora.Web
	web.Get("/test", func() {
		fmt.Println("test")
	})
	aurora.Run(web)
}

WARNING

aurora 需要 >= v0.3.8
